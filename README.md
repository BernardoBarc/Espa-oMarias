Site criado com React, node js e banco de dados mongodb.
Este site tem como funcionalidade principal, solicitar agendamento por parte do cliente com o(a) profissional de sua escolha.
    A manicure que for escolhida por parte do cliente tem total permissão de aceitar o atendimento ou então cancelar o pedido de agendamento, caso a manicure aceite o horário, ela verá o horário na aba de horários confirmados, caso seja finalizado tem a opção de marcar finalizado, ou então cancelar, caso a manicure queira ou mudar o dia do agendamento ou não puder atender.
O projeto conta também com funcionalidades, em que o usuário admin tem total capacidade de modificação, como por exemplo, fotos na página inicial, dados do footer, imagens com titulo e descrição de serviços praticados no ambiente de trabalho.
Além de tudo o projeto conta com uma estilização super responsiva, moderna e com cores atrativas.
